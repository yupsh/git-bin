# yup-git

```
NAME:
   git - git command wrapper for yupsh

USAGE:
   git [OPTIONS] COMMAND [ARG...]

      Execute git commands.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --repository value         repository path or URL
   --branch value, -b value   branch name
   --remote value             remote name
   --message value, -m value  commit message
   --author value             author name
   --email value              author email
   --force, -f                force operation (default: false)
   --verbose, -v              be verbose (default: false)
   --quiet, -q                be quiet (default: false)
   --dry-run, -n              dry run (default: false)
   --all, -a                  all files (default: false)
   --interactive, -i          interactive mode (default: false)
   --help, -h                 show help
```
